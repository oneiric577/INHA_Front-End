# 🧑‍💻 My Portfolio Website

나의 이력, 자기소개, 프로젝트 등을 담은 개인 포트폴리오 웹사이트입니다.  
Front-end 수업에서 배운 기술을 사용하여 제작하였습니다.

## 📌 주요 내용

- 자기소개
- 이력서 요약
- 프로젝트 포트폴리오
- 연락처 및 소셜 링크

## 🛠 사용 기술

- HTML
- CSS
- JavaScript
- Cursor
